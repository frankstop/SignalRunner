# Signal Runner for macOS

**Signal Runner** is a small macOS app build packaged for local testing.

This release is intended as a **test build**, not a fully notarized public macOS release.

---

## Requirements

- macOS 14.0 or newer
- Apple Silicon Mac

---

## Installation

1. Download `SignalRunner-macOS.zip`.
2. Double-click the zip file to extract it.
3. Move `Signal Runner.app` into your **Applications** folder.
4. Open `Signal Runner.app`.

---

## Opening the App

Because this is a direct test build, macOS may warn that the app is from an unidentified developer.

If macOS blocks the app:

1. Right-click `Signal Runner.app`.
2. Choose **Open**.
3. Click **Open** again when prompted.

You can also allow it through macOS settings:

1. Open **System Settings**.
2. Go to **Privacy & Security**.
3. Scroll down to the blocked app message.
4. Click **Open Anyway**.

---

## Notes

This app is not currently notarized through Apple, so the macOS security warning is expected.

Use this build for testing only.
